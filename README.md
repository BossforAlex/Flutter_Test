# Flutter 高德导航广播监听

这是一个基于 Flutter 开发的应用程序，主要功能是实现对高德地图自动导航功能的广播监听。

## 项目简介

本项目通过 Flutter 和原生 Android 开发相结合的方式，实现了以下功能：

- 监听高德地图自动导航广播信息
- 处理导航数据并在应用中展示
- 支持实时接收导航状态更新

## 技术栈

- Flutter: 跨平台UI开发框架
- Kotlin: Android原生开发
- 高德地图SDK: 导航功能支持

## 开始使用

1. 确保您的开发环境已安装以下工具：
   - Flutter SDK
   - Android Studio
   - Kotlin 开发环境

2. 克隆项目到本地：
   ```bash
   git clone https://github.com/BossforAlex/Flutter_Test.git
   ```

3. 安装项目依赖：
   ```bash
   flutter pub get
   ```

4. 运行项目：
   ```bash
   flutter run
   ```
