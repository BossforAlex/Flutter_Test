# 发布说明（中文）

## 版本 3.2.0 - 重新开始

本次发布为项目重新启动后的首个版本，聚焦于修复编译问题、优化前端 UI 和统一中文发布说明。

### 修复内容
- **编译问题修复**：移除冗余的 `build.gradle.kts` 配置文件，避免与 Groovy 版本冲突。
- **测试修复**：更新 `widget_test.dart` 以匹配当前 UI 结构（标题、底部导航标签）。

### 发布说明
- 后续所有发布说明将仅使用中文，不再包含英文翻译。

### 功能概述
- 高德地图导航监听：实时接收并展示导航广播数据，支持数据去重、缓存管理、调试工具。
- 蓝牙控制：基于 `flutter_blue_plus` 实现真实蓝牙设备扫描与连接。
- 底部导航栏：采用 `IndexedStack` 保持页面状态，三个标签页分别为"导航监听"、"蓝牙控制"、"设置"。

### 使用建议
- 导航监听调试：进入"导航监听"页 → 点击右上角"调试"按钮 → 发送测试广播以验证链路。
- 蓝牙扫描：确保系统蓝牙与定位服务已开启，进入"蓝牙控制"页 → 点击"开始扫描"。
